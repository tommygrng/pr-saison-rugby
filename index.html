🚀 Déployer sur GitHub Pages

  1. Compte gratuit sur github.com
  2. Nouveau dépôt public (ex : preseason-rugby)
  3. Add file → Upload files → glisser index.html → Commit
  4. Settings → Pages → Source: main / root → Save
  5. URL : https://PSEUDO.github.io/preseason-rugby/

💾 Synchroniser entre appareils

Données stockées dans le navigateur. Pour partager : Export → envoyer le .json → Import sur l'autre appareil. La config (références, structure semaines) est incluse dans l'export.

📱 Mobile

Safari : partage → Ajouter à l'écran d'accueil. Android Chrome : ⋮ → Ajouter à l'écran d'accueil.
