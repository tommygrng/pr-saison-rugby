🚀 Déployer cette application gratuitement en 5 minutes

Ce fichier index.html est 100% autonome : pas de serveur, pas de base de données. Il s'héberge gratuitement sur GitHub Pages et fonctionne sur tous les navigateurs (ordinateur, tablette, téléphone).

1

Créez un compte gratuit sur github.com si vous n'en avez pas.

2

Cliquez sur « New repository », nommez-le par exemple preseason-rugby, cochez Public, puis cliquez Create repository.

3

Dans le dépôt créé, cliquez « Add file » → « Upload files », glissez-déposez le fichier index.html, puis cliquez Commit changes.

4

Allez dans Settings → Pages (menu gauche). Sous Source, sélectionnez la branche main et le dossier / (root), puis cliquez Save.

5

Après ~1 minute, votre application est disponible à l'adresse :
https://VOTRE-PSEUDO.github.io/preseason-rugby/

💾 Sauvegarde et partage des données

  • Les données sont sauvegardées automatiquement dans le navigateur (localStorage) sur chaque appareil.
  • Pour partager les données entre plusieurs appareils ou staff : utilisez Exporter (génère un fichier .json) et Importer sur l'autre appareil.
  • En option avancée, vous pouvez héberger sur Netlify Vercel pour obtenir une URL propre et personnalisée.

📱 Utilisation sur mobile

  • Ouvrez l'URL dans Safari (iOS) ou Chrome (Android).
  • Appuyez sur le bouton partage → « Ajouter à l'écran d'accueil » pour une icône comme une vraie app.

🔄 Mettre à jour les données de planification

Toutes les cibles sont dans la section DATA au début du script JavaScript. Pour les modifier, ouvrez le fichier avec un éditeur de texte (Notepad, TextEdit, VS Code) et ajustez les valeurs, puis ré-uploadez sur GitHub.
