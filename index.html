🚀 Déployer sur GitHub Pages (gratuit)

Ce fichier index.html est 100% autonome. Hébergement gratuit sur GitHub Pages.

  1. Créez un compte sur github.com
  2. Nouveau dépôt public, nommé par ex. preseason-rugby
  3. Add file → Upload files → glissez index.html → Commit
  4. Settings → Pages → Source : branche main / root → Save
  5. URL : https://PSEUDO.github.io/preseason-rugby/

💾 Synchroniser entre appareils

Les données sont stockées dans le navigateur (localStorage). Pour partager entre staff : Exporter → envoyer le .json → Importer sur l'autre appareil.

📱 Ajouter à l'écran d'accueil (mobile)

Safari iOS : bouton partage → Ajouter à l'écran d'accueil. Android Chrome : menu ⋮ → Ajouter à l'écran d'accueil.
